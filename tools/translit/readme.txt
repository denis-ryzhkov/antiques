Транслитерирует имена ВСЕХ файлов в текущей папке и её подпапках.

Написана чтобы подготавливать книги с русскими именами файлов
для прочтения ReadManiac-ом на мобильном телефоне Samsung SGH-D780-DUOS
(на Nokia проблем с русскими именами файлов не было).
http://sourceforge.net/projects/readmaniac
