*) Database file db.txt line format:
dd.MM.yyyy name

example:
31.01.1970 Bob Smith
29.02.0000 Nick Happy
23.08.1982 Creon

Year 0000 means "unknown".

*) To prevent "No coming birthdays found." dialog,
for example at startup, use "-s" command line switch.
